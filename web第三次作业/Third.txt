（1）<input type ="text/"> 单行文本输入框
（2）<input type = "password/"> 用来输入密码，内容①圆点的形式现实
（3）<input type ="radio/"> 单选按钮，可以设置默认选中属性：checked
（4）<input type = "submit">  提交表单数据按钮，可以通过设置value属性改变按钮上的默认文本
（5）<input type ="reset"/>  单机充值按钮取消已输入表单信息
（6）<input typr = "file"/> 选择文件按钮，可以通过单机按钮后直接选择文件，将文件提交给后台服务器
